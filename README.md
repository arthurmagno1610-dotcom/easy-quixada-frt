# EasyQuixadá Admin

Projeto web estático gerado a partir do layout do Figma do painel administrativo do EasyQuixadá.

## Como abrir

Abra o arquivo `index.html` diretamente no navegador.

## Estrutura

- `index.html`: ponto de entrada da página.
- `src/main.js`: renderização das telas e navegação interna.
- `src/styles.css`: estilos responsivos do painel.

## Telas

- Dashboard com sidebar, painel de boas-vindas, cards de ação e bloco de dica rápida.
- Lojas com métricas, busca, lista de lojas e formulário de cadastro.
- Demais módulos com estados preparados para expansão.

## Cadastro de lojas

O botão `Nova Loja` abre um formulário com:

- ID do estabelecimento
- Descrição
- Nome do estabelecimento
- Endereço
- Telefone
- Site
- Imagem de capa
- Instagram
- Facebook
- WhatsApp
- Categoria

Como o projeto é estático, os cadastros ficam em memória enquanto a página está aberta.
